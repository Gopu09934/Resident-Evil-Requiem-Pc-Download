# Resident Evil Requiem - Optimized Desktop Version

**Resident Evil Requiem** is the latest mainline entry in the iconic survival horror series by Capcom.

Step into a gripping story where the paths of **Grace Ashcroft** and legendary agent **Leon S. Kennedy** intertwine in a terrifying new chapter filled with suspense, intense combat, and unforgettable moments.

This repository contains a highly optimized PC build designed for modern Windows systems.

## ✨ Features

- Optimized build for Windows 10 and Windows 11
- Excellent performance on a wide range of hardware
- Support for 1080p, 1440p, 4K and ultrawide resolutions
- Improved keyboard + mouse controls
- Full gamepad support (Xbox / PlayStation)
- Portable-ready structure
- Pre-tuned configuration files for better stability and visuals

## 🚀 Installation Guide

1. Download the archive from the Releases tab
2. Extract the files using WinRAR or 7-Zip
3. Run `RE_Requiem.exe` as administrator
4. Adjust graphics settings in-game according to your hardware
5. Enjoy the experience!

**Tip:** It is recommended to add the game folder to your antivirus exclusions to avoid false positives.
